# Onboarding Modal Design Notes

Based on the reference images, the onboarding flow has 3 steps:

## Step 1: Welcome / Journey Start (Image 3)
- Title: "开启你的AI冒险之旅" (Start your AI adventure journey)
- Subtitle: "您是否一直想了解人工智能的功能？或者学习一种新的人工智能工具并充分利用它？"
- Description: "我们的指南将帮助您确定哪款工具最适合您，最符合您的目标。为了方便起见，每份指南都将重点介绍一种特定的AI工具。"
- Visual: Circular layout with AI tool icons orbiting around a central star icon
- Progress bar at top
- Close (X) button

## Step 2: Course Selection (Image 2)
- Title: "在我们深入探讨之前......" (Before we dive in...)
- Subtitle: "您最有兴趣首先探索哪种人工智能工具？" (Which AI tool are you most interested in exploring first?)
- List of course options with icons:
  - ChatGPT - 探索高级提示技巧
  - Jasper AI - 学会改变你的写作方式
  - 旅程中途 (Midjourney) - 利用人工智能探索令人叹为观止的艺术作品
  - 可爱的 (Lovable) - 利用人工智能即时创建网站
  - 双子座 (Gemini) - 设置多步骤工作流程
- Each option is a clickable card with icon, title, and description

## Step 3: Plan Ready (Image 1)
- Title: "你的计划已准备就绪！" (Your plan is ready!)
- Subtitle: "让我们深入了解一下成为人工智能专家并将这些强大的工具融入日常生活所需的步骤。"
- Shows course path with modules:
  - "人工智能精通" (AI Mastery) - 28个模块
  - Grid of course modules with icons
  - "You're here" indicator on first module
  - Locked modules shown with lock icons
- CTA Button: "我们走吧！" (Let's go!)

## Design Elements
- Dark modal background (gray/dark theme)
- Purple accent color for buttons and progress bar
- Rounded corners on cards
- Clean, modern typography
- Progress indicator at top
- Close button (X) in top left
