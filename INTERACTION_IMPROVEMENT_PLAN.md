## 六、点击交互实现技术方案

### 6.1 Weekly Streaks 弹窗实现

```
组件结构：
- StreakModal.tsx（模态框容器）
  - StreakHeader（火焰动画 + 当前天数）
  - StreakChallenge（进度条：Day 0 → 3 → 7 → 14）
  - StreakStats（当前连续/最长连续）
  - StreakCalendar（月历视图）

技术要点：
1. 使用 Framer Motion 实现火焰动画
2. 使用 CSS Grid 实现月历布局
3. 使用 React Portal 实现模态框
4. 点击背景或 X 按钮关闭
```

### 6.2 模块预览弹窗实现

```
组件结构：
- ModulePreviewPopover.tsx（弹窗容器）
  - ModuleTitle（模块标题）
  - ModuleDescription（模块描述）
  - ActionButtons（Read/Listen 按钮）

技术要点：
1. 使用 Radix UI Popover 或自定义实现
2. 弹窗位置跟随点击的模块图标
3. 点击外部区域关闭弹窗
4. 支持键盘导航（Escape 关闭）
```

### 6.3 课程切换下拉实现

```
组件结构：
- CourseDropdown.tsx（下拉容器）
  - DropdownTrigger（当前课程名 + 箭头）
  - DropdownContent（横向滚动的课程列表）
    - CourseItem（课程图标 + 名称 + 进度）

技术要点：
1. 使用 overflow-x: auto 实现横向滚动
2. 当前课程有蓝色边框高亮
3. 点击课程项触发路由跳转
4. 使用 CSS scroll-snap 优化滚动体验
```

### 6.4 深色模式实现

```
技术方案：CSS Variables + React Context + localStorage

步骤：
1. 定义颜色变量（:root 和 [data-theme="dark"]）
2. 创建 ThemeContext 管理主题状态
3. 使用 localStorage 持久化用户偏好
4. 更新所有组件使用 CSS 变量

颜色变量示例：
:root {
  --bg-primary: #ffffff;
  --bg-secondary: #f5f5f5;
  --text-primary: #1a1a1a;
  --text-secondary: #666666;
}

[data-theme="dark"] {
  --bg-primary: #1a1a2e;
  --bg-secondary: #2d2d44;
  --text-primary: #ffffff;
  --text-secondary: #a0a0a0;
}
```

## 八、风险与注意事项

### 8.1 主要风险

| 风险 | 影响 | 缓解措施 |
|-----|------|---------|
| 弹窗实现复杂 | 开发周期延长 | 使用成熟 UI 库（Radix UI） |
| 深色模式影响范围大 | 可能引入视觉 bug | 分组件逐步适配 |
| 动画性能问题 | 用户体验下降 | 使用 CSS 动画优先，避免 JS 动画 |

### 8.2 注意事项

1. **渐进式改进**：不要一次性实现所有交互，分迭代逐步完善
2. **保持功能稳定**：交互改进不应影响现有功能
3. **性能监控**：新增动画可能影响性能，需要监控
4. **无障碍支持**：确保弹窗、下拉菜单支持键盘导航和屏幕阅读器
