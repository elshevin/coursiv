## 五、技术实现说明

### 路由结构
```
/                    → 主落地页
/onboarding          → Quiz 流程入口
/onboarding/:step    → Quiz 各步骤页面
/upsell              → 付费升级页
/blog                → 博客列表
/blog/:slug          → 博客详情
/support             → 支持中心
```

### 数据库表
```
users          → Manus OAuth 用户（已有）
demo_users     → Demo 测试用户（已创建）
quiz_responses → Quiz 答案存储（待创建，可选）
```

## 六、交付标准

1. **视觉还原度**：与 Figma 设计稿 100% 一致（桌面端优先）
2. **功能完整性**：所有页面可访问，所有交互可操作
3. **Mock 数据**：生成式内容使用静态数据填充
4. **Demo 登录**：每次登录创建新用户，存储到数据库
5. **无 Bug**：主流程无阻塞性 Bug
